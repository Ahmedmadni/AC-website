# AC-website

موقع إلكتروني للإعلان عن خدمات بيع وتركيب وصيانة المكيفات داخل مدينة الرياض، المملكة العربية السعودية.

- الهاتف / واتساب: 0560409811

## بنية الملفات

```
index.html      الصفحة الرئيسية (كل أقسام الموقع)
css/style.css   تنسيقات إضافية فوق Tailwind CDN
js/main.js      القائمة الجوالة، الأسئلة الشائعة، وإرسال نموذج التواصل عبر واتساب
```

## التقنيات المستخدمة

- HTML5 + Tailwind CSS (عبر CDN، بدون أدوات بناء)
- خط Tajawal من Google Fonts (دعم كامل للغة العربية RTL)
- أيقونات Font Awesome
- جافاسكربت خام بدون أي إطار عمل — الموقع ثابت بالكامل ويمكن استضافته على أي منصة (GitHub Pages, Netlify, Vercel, Lovable...)

## المعاينة محلياً

يكفي فتح `index.html` مباشرة في المتصفح، أو تشغيل خادم بسيط:

```bash
python3 -m http.server 8080
```

## التخصيص

- تغيير اسم الشركة والشعار: ابحث عن "فريق تكييف الرياض" في `index.html` واستبدله باسمك التجاري.
- تغيير رقم الهاتف/واتساب: ابحث عن `0560409811` و`966560409811` واستبدلهما.
- الألوان: عدّل ألوان `primary` و`accent` داخل إعداد Tailwind في أعلى `index.html`.
